Bakku Runner

Jump: Press the up arrow key "↑" (Hold to jump higher)

Description

Start playing Bakku Runner. Jump over to avoid enemies like raccoons, wolve, or polar bears to get high score.

Bakku Runner features:

3 ENEMIES

-> Raccoon -> Wolf -> Polarbear

3 GAME MODES

• Easy: Change background to woodland, and racoons show up

• Medium: Change background to city, and wolves show up

• Hard: Change background to winter, and polarbears show up

Have fun playing Bakku Runner!
